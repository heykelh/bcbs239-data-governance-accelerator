# Data Roles & RACI

## Main roles

### Data Owner
Business representative accountable for the meaning, usage, quality expectations and prioritization of remediation for critical data.

### Data Steward
Operational role responsible for monitoring quality indicators, maintaining data definitions and coordinating issue remediation.

### IT Custodian
Technical role responsible for pipelines, storage, controls implementation and platform stability.

## Simplified RACI

| Activity | Data Owner | Data Steward | IT Custodian |
|---|---|---|---|
| Define business rules | A | R | C |
| Monitor DQ KPIs | C | R | C |
| Implement technical controls | C | C | R |
| Validate remediation priority | A | R | C |
| Update data dictionary | C | R | C |
| Escalate critical issues | A | R | C |

## Interpretation

The model intentionally distinguishes:
- business accountability
- operational quality coordination
- technical implementation responsibility

This separation helps avoid ambiguity and supports faster remediation of critical issues.
